# Connect Canvas Constitution

## Principles

1. **Human dignity first** — language should support agency, nuance, and care.
2. **Privacy by default** — collect the minimum data needed and keep sensitive moments protected.
3. **Mobile-first clarity** — every screen should work well on a phone before expanding to larger viewports.
4. **Deployable increments** — the app should remain safe to deploy after every commit.
5. **Foundation before features** — technical structure comes before Discovery or business workflows.

## Engineering standards

- Use the Next.js App Router with TypeScript.
- Style with Tailwind CSS and shadcn/ui-compatible primitives.
- Use Drizzle ORM with Neon PostgreSQL for database access.
- Keep linting and build checks available through package scripts.
