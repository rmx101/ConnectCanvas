# Connect Canvas Roadmap

## S1 — Application foundation

- Replace the static placeholder with a Next.js App Router shell.
- Add TypeScript, Tailwind CSS, shadcn/ui structure, Drizzle ORM, Neon PostgreSQL wiring, and ESLint.
- Preserve Vercel deployability.

## Later slices

- Define the Discovery flow.
- Add database-backed domain models.
- Build guarded user journeys.
- Add observability, privacy reviews, and production hardening.

Discovery work must not begin until the foundation is accepted.
